# Eat-Da-Burger
 
**Project Description** - Eat-Da-Burger is a full-stack web application that allows the user to input burgers they would like to eat, select which ones they have eaten, and remove them from the list The purpose of this assignment is to display my growing abilities to develop a full-stack application.

## Application Heroku Link
The link below will give you direct access to Eat-Da-Burger web application using your web browser via the Heroku web service. (NOTE: There will be a momentary delay when first accessing the Heroku servers.)

https://still-spire-95543.herokuapp.com/

## Local Environment Setup
To use Eat-Da-Burger web application application from your local environment, you must accomplish the following steps below:

**Step 1 - Clone my repo using the command line below.**
```
git clone git@github.com:kmj217/Eat-Da-Burger.git
```
**Step 2 - Change directory to the cloned repo folder.**
```
cd Eat-Da-Burger
```
**Step 3 - Install all required NPM packages.**
```
npm install
```
**Step 4 - Start the application server using the command line below**
```
node server.js
```
